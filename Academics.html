Arduino Project

The video you see below is of a final project that I and my team made in our CSIS 2810 (computer architecture) class. Together we designed a self-maintained terrarium/greenhouse. In this project we used arduino kits containing various small electronics, which could all be controlled using an arduino board. We had to come up with the design and program the arduino board so it could work and communicate with the various components, like the humidity sensor, the fan, the water pump, LCD screen, buttons, etc. The user can set the watering cycles, the humidity levels, the light levels, according to the needs of the plants you keep inside.
